Bootcamp 2021: Art Generator

My first programming project, concluding the 2 week Programming Bootcamp at City (2020). This Processing project demonstrates a understanding of the use of methods, loops, and randomness. I created an random art generator in which it will produce one of my three designs randomly. The designs being circles, triangles and squares. Each time the program is laoded it will randomise the colours that appear giving you a unique design each time. Using keyPressed I have allowed it to manually change the colour of the a particular set of shapes independently from the rest using keys 1-8. These colours are also random however.

Check out the code on Github: https://github.com/Tooba52/adck772
